# CATech Engine

Water bottle model created by Khronos: <https://github.com/KhronosGroup/glTF-Sample-Assets>

## Running the demo

### Unix

```sh
cd path/to/catech-engine/bin
./catech-engine
```

#### Darwin/macOS

```sh
cd path/to/catech-engine/bin
./catech-engine-macos.sh
```

### Windows

```bat
cd path\to\catech-engine\bin
catech-engine.bat
```
